Problem 17 — 1.4 Exercises (Determining Continuity)

Determine the interval on which the function \(\displaystyle \frac{3}{5 - x^3} \) is continuous.

SOLUTION We wish to avoid any points at which the denominator evaluates to \(0\). Setting the denominator equal to \(0\), we see that \(x = \sqrt[3]{5}\) puts the function into a \(3/0\) form. Therefore, we must exclude it. The function is continuous for all other \(x\), however, as given by IV. of Theorem \(2\)—the functions \(y = 3\) and \(y = 5 - x^3\) are continuous for all \(x\), but we must exclude \(x = \sqrt[3]{5}\) to avoid division by zero. The interval is then \[(-\infty, \sqrt[3]{5}) \cup (\sqrt[3]{5}, \infty). \]
