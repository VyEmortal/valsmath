Problem 22 — 1.4 Exercises (Determining Continuity)

Determine the interval on which the function \(\displaystyle \frac{\sqrt{12 - 5x}}{x^4 + 16} \) is continuous.

SOLUTION When dealing with a quotient of functions, we seek the values \(x\) for which the numerator and denominator are both continuous but the denominator does not equal \(0\). In this case, the numerator, \(\sqrt{12 - 5x}\), is continuous for all \(x\) in its domain; i.e., for \(x \leq 12/5\). Furthermore, the denominator, \(x^4 + 16\), is a polynomial, and polynomials are continuous for all values of \(x\).

We now want the values of \(x\) for which the denominator equals zero so that we can exclude them. However, in this case, attempting to solve the equation \(x^4 + 16 = 0\) yields no real values of \(x\). Therefore, the denominator will never evaluate to zero.

We therefore have that, for \(x \leq 12/5\), the numerator and denominator are both continuous and that there are no issues with division by zero. Then, IV. of Theorem \(2\) assures us that the quotient \(\displaystyle \frac{\sqrt{12 - 5x}}{x^4 + 16} \) will be continuous for all \(x\) in \[ \left(-\infty, \frac{12}{5}\right).\]
