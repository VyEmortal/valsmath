Problem 3 — 1.4 Exercises (Determining Continuity)

Determine whether \(3^{x + 4}\) is continuous at \(x = 2\).

SOLUTION Using exponent laws, we can rewrite the function as \[ \begin{aligned} 3^{x + 4} &= 3^{x} \cdot 3^4 \\ &= 81(3)^x. \\ \end{aligned} \] An exponential function with a positive base not equal to \(1\) is continuous for all real numbers. Therefore, \(3^{x + 4}\) is continuous at \(x = 2\).
