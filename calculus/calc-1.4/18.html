Problem 18 — 1.4 Exercises (Determining Continuity)

Determine the interval on which the function \(\displaystyle \sin(x) + 6^x \) is continuous.

SOLUTION Because the functions \(\sin(x)\) and \(6^x\) are both continuous for all \(x\), the sum \(\sin(x) + 6^x\) is also continuous for all \(x\), as given by I. of Theorem \(2\). The interval is then \[-\infty \lt x \lt \infty. \]
