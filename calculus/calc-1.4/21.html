Problem 21 — 1.4 Exercises (Determining Continuity)

Determine the interval on which the function \(\displaystyle \frac{\log(x) - 4}{x^2 - 8} \) is continuous.

SOLUTION We want to find the values of \(x\) for which \(\log(x) - 4\) and \(x^2 - 8\) are both continuous and \(x^2 - 8 \ne 0\).

We can rewrite the numerator, using logarithm properties, as \[ \begin{aligned} \log(x) - 4 &= \log(x) - \log(10^4) \\ &= \log \left(\frac{x}{10^4}\right). \end{aligned} \] We understand that logarithms are continuous for all values in their domain. In this case, the domain of \(\displaystyle \log \left(\frac{x}{10^4}\right)\) includes all positive values of \(x\); i.e., \(x \gt 0\).

On the denominator, \(x^2 - 8\) is a polynomial and is therefore continuous for all \(x\). However, we must avoid any instances in which this expression equals \(0\). \[ \begin{aligned} x^2 - 8 &= 0 \\ x = -2 \sqrt{2} \quad &\textrm{and} \quad x = 2 \sqrt{2}. \end{aligned} \]

Therefore, the interval on which \(\displaystyle \frac{\log(x) - 4}{x^2 - 8} \) is continuous is \[(0, 2 \sqrt{2}) \cup (2 \sqrt{2}, \infty). \]
