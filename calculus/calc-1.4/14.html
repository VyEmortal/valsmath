Problem 14 — 1.4 Exercises (Determining Continuity)

Let \( k(x)= \begin{cases} e^{2x} + 7 &x \leq q \\[2ex] 5e^x + 1 &x \gt q. \end{cases} \)

Find the values of \(q\) for which \(k(x)\) is continuous at \(x = q\).

SOLUTION In order for \(k(x)\) to be continuous at \(x = q\), \(\lim_{x \to q} k(x)\) must equal \(k(q)\).

We understand that \(k(x) = e^{2x} + 7\) for \(x \leq q\). Therefore, the left-hand-side limit, \(\lim_{x \to q^-} k(x)\), equals \(k(q) = e^{2q} + 7\). Furthermore, we consider the case in which \(k(x)\) is continuous, so \(\lim_{x \to q^+} k(x)\) must also equal \(k(q)\). We therefore set up the equation \[ \begin{aligned} e^{2q} + 7 &= 5e^q + 1 \\ e^{2q} - 5e^q + 6 &= 0, \\ \end{aligned} \] which is analogous to a quadratic (you may perform the substitution \(u = e^q\) if the form is unclear) in that it can be factored as \[(e^q - 3)(e^q - 2).\] Solving for \(q\), we set each factor equal to \(0\) and utilize logarithms to obtain \[ \begin{aligned} e^q - 3 &= 0 \\ e^q &= 3 \\ q &= \ln(3). \\ \end{aligned} \] \[ \begin{aligned} e^q - 2 &= 0 \\ e^q &= 2 \\ q &= \ln(2). \\ \end{aligned} \] Therefore, \(q = \ln(2)\) and \(q = \ln(3)\) permit \(k(x)\) to be continuous at \(x = q\).
