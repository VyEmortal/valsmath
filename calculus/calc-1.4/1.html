Problem 1 — 1.4 Exercises (Determining Continuity)

Consider Fig. 1.4-4., in which the graph of \(f(x)\) is shown. Points \(A(-5, 4)\), \(B(-2.5, -5)\), and \(C(8, 3)\) are labeled, with point \(B\) being the location of a removable discontinuity. Determine whether \(f(x)\) is continuous at each point.

SOLUTION For this problem, we see that point \(A(-5, 4)\) is an endpoint. Because it is a left endpoint, to determine whether \(f(x)\) is continuous at point \(A\), we only need to verify that \[ \lim_{x \to -5^+}f(x) = f(-5).\] In this case, we see that \(f(-5) = 4\) and \(\lim_{x \to -5^+} = 4\). Because \(\lim_{x \to -5^+}f(x) = f(-5)\), \(f(x)\) is continuous at point \(A\).

For point \(B(-2.5, -5)\), we notice that \(f(x)\) is not defined there (there is a removeable discontinuity) and is therefore discontinuous at that point.

For point \(C(8, 3)\), we notice that \(f(8) = 3\) and \(\lim_{x \to 8} f(x) = 3\). Because \(\lim_{x \to 8} f(x) = f(8) = 3\), we determine that \(f(x)\) is continuous at that point.
