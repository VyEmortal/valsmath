1.4 — Determining Continuity

In this section, we will define continuity and learn how to determine it. We have briefly discussed the idea of continuity in previous sections, but it is appropriate to formally define it using limits and connect the definition to our prior experiences.

If a function is continuous at a certain point, then we can trace that portion of the graph without lifting our pen; i.e., we could draw it in one step. Formally, a function \(f(x)\) is continuous at \(x = a\) if \(\lim_{x \to a} f(x) = f(a).\) It is important to note, however, that \(\lim_{x \to a} f(x)\) may exist, but not equal \(f(a).\) This scenario usually arises when a function \(f(x)\) contains a hole (formally called a "removable discontinuity") at \(x = a.\)


DEFINITION OF CONTINUITY

If \(\displaystyle \lim_{x \to a} f(x) = f(a),\) then \(f(x)\) is continuous at \(x = a.\)


As an example, consider the graph of \(f(x),\) shown in Fig. 1.4-1.


From the graph, we observe that \(f(x)\) contains a removable discontinuity at the labeled point \(A(-4, -3),\) and we note that \(\lim_{x \to -4} f(x) = -3.\) However, because \(f(-4) \ne -3\) (it is undefined), we conclude that \(f(x)\) is discontinuous at \(x = -4.\) Conversely, at point \(B(2,2),\) we notice that \(\lim_{x \to 2} f(x) = 2\) and \(f(2) = 2.\) Therefore, since \(\lim_{x \to 2} f(x) = f(2),\) the function is continuous at \(x = 2.\) This result is intuitive because, at point \(B,\) we can trace the graph without needing to lift our pen. At point \(C(8, 10),\) which is an endpoint, we can establish continuity from the left because \(\lim_{x \to 8^-} f(x) = f(8),\) but we cannot establish continuity from the right (we cannot take a right-hand-side limit.) Therefore, the interval of \(x\)-values for which \(f(x)\) is continuous is \[ (-\infty, -4) \cup (-4, 5) \cup(5, 8]. \] Note that we include \(8\) because it is an endpoint. For a right endpoint, we understand that "continuous" implies "continuous from the left."


THEOREM \(1\): CONTINUITY OVER AN INTERVAL

If a function \(f(x)\) is continuous at every point in an interval \([a,b],\) then \(f(x)\) is continuous on that interval. For the endpoints \(a\) and \(b,\) it is implied that \(f(x)\) is continuous from the right and left, respectively.


STEPS FOR DETERMINING CONTINUITY

To determine whether \(f(x)\) is continuous at \(x = a\):

\((1)\) Ensure that \(f(x)\) is defined at \(x = a.\)

\((2)\) Check the one-sided limits to establish that \(\displaystyle \lim_{x \to a} f(x)\) exists.

\((3)\) Verify that \(\displaystyle \lim_{x \to a} f(x) = f(a).\)

For endpoints, it is sufficient to only consider the appropriate one-sided limit. If \(a\) is a left endpoint, then \(f(x)\) is continuous at \(x = a\) if \(\displaystyle \lim_{x \to a^+} f(x) = f(a).\) If \(a\) is a right endpoint, then \(f(x)\) is continuous at \(x = a\) if \(\displaystyle \lim_{x \to a^-} f(x) = f(a).\)


EXAMPLE \(1\)

Consider Fig. 1.4-2., in which the graph of \(g(x)\) is shown. Points \(A(-1,8),\) \(B(6,5),\) and \(C(12,5)\) are labeled, with point \(A\) being the location of a jump discontinuity. Determine whether \(g(x)\) is continuous at each point.

At point \(A,\) we notice that \(g(x)\) is undefined. Thus, \(g(x)\) is not continuous at that point.

At point \(B,\) we notice that \(g(6) = 5.\) We can also verify that \[\lim_{x \to 6} g(x) = 5. \] Therefore, because \(\lim_{x \to 6} g(x) = g(6) = 5,\) we establish that \(f(x)\) is continuous at point \(B.\)

We notice that point \(C\) is an endpoint. We cannot take the right-hand-side limit, but we notice that \(\lim_{x \to 12^-} g(x) = g(12) = 5.\) This statement is sufficient to establish that \(g(x)\) is continuous at point \(C.\)


Let us now discuss piecewise functions and algebraically determining continuity. Consider, for example, the piecewise function \(f,\) defined by \[ f(x)= \begin{cases} -3 \cos(x) &x \leq 0 \\[2ex] 2x - 5 &x \gt 0. \end{cases} \] In order for \(f(x)\) to be continuous at \(x = 0,\) \(f(0)\) must be defined and \(\lim_{x \to 0} f(x)\) must equal \(f(0).\) We notice that \(f(0)\) is defined and equals \(-3 \cos(0) = -3.\) We now determine whether \(\lim_{x \to 0} f(x)\) matches this value, \(-3.\) We notice that, as \(x\) approaches \(0\) from the left-hand side, \(f(x)\) approaches \(-3 \cos(0) = -3.\) Furthermore, as \(x\) approaches \(0\) from the right-hand side, \(f(x)\) approaches \(2(0) - 3 = -5.\) Therefore, because the one-sided limits are not equivalent, \(\lim_{x \to 0} f(x)\) does not exist. Since \(\lim_{x \to 0} f(x) \ne f(0),\) we conclude that \(f(x)\) is discontinuous at \(x = 0.\) (In this case, \(f(x)\) contains a jump discontinuity at \(x = 0,\) as shown in Fig. 1.4-3.)


EXAMPLE \(2\)
  1. Let \( \displaystyle f(x)= \begin{cases} x^2 - 4x &x \lt 2 \\[2ex] -2^x &x \geq 2. \end{cases} \)

    Determine whether \(f(x)\) is continuous at \(x = 2.\)

  2. Let \( \displaystyle g(x)= \begin{cases} \sin(x) &x \lt \pi/2 \\[2ex] 0 \, &x \geq \pi/2. \end{cases} \)

    Determine whether \(g(x)\) is continuous at \(x = \pi/2.\)

  3. Let \( \displaystyle v(x)= \begin{cases} x^2 + 4 &x \lt -4 \\[2ex] 2^{-x} + 4 &x \gt -4. \end{cases} \)

Determine whether \(v(x)\) is continuous at \(x = -4.\)

  1. We see that \(f(2)\) is defined and equals \(-4.\) Now, we will determine whether \(\lim_{x \to 2} f(x)\) matches this value, \(-4.\) We must check the one-sided limits and establish that they are equivalent. We see that, as \(x\) approaches \(2\) from the left (i.e., \(x \lt 2\)), \(f(x)\) approaches \((2)^2 - 4(2) = -4.\) Furthermore, as \(x\) approaches \(2\) from the right (i.e., \(x \geq 2\)), \(f(x)\) approaches \(-2^2 = -4.\)

    Therefore, because the one-sided limits are equivalent and equal to \(-4,\) \(\lim_{x \to 2} f(x) = f(2) = -4.\) Therefore, \(f(x)\) is continuous at \(x = 2.\)

  2. In order for \(g(x)\) to be continuous at \(x = \pi/2,\) we must establish that \(g(\pi/2)\) is defined and that \(\lim_{x \to \pi/2} g(x) = g(\pi/2).\)

    We see that \(g(\pi/2) = 0.\) Now, checking the one-sided limits, we observe that, as \(x\) approaches \(\pi/2\) from the left (i.e., \(x \lt \pi/2\)), \(g(x)\) approaches \(\sin(\pi/2) = 1.\) Furthermore, as \(x\) approaches \(\pi/2\) from the right (i.e., \(x \gt \pi/2\)), \(g(x)\) approaches \(0.\)

    Because the one-sided limits are not equivalent, \(\lim_{x \to \pi/2} g(x)\) does not exist. Therefore, \(g(x)\) is not continuous at \(x = \pi/2.\)

  3. We notice that \(v(x)\) is not defined at \(x = -4.\) Therefore, \(v(x)\) is not continuous at \(x = -4.\)



EXAMPLE \(3\)

Let \( \displaystyle f(x)= \begin{cases} kx + 4 &x \leq 3 \\[2ex] 2k - 5 &x \gt 3. \end{cases} \)

Find the values of \(k\) for which \(f(x)\) is continuous at \(x = 3.\)

For this problem, we notice that \(f(3) = 3k + 4.\) Therefore, \(\lim_{x \to 3} f(x)\) must exist and match this value, equaling \(f(3) = 3k + 4,\) in order for \(f(x)\) to be continuous at \(x = 3.\) Now, we will take the one-sided limits and set them equal to each other, and verify that they equal \(f(3).\) \[ \begin{aligned} \lim_{x \to 3^-} f(x) &= 3k + 4. \\ \lim_{x \to 3^+} f(x) &= 2k - 5. \\ \end{aligned} \] \[ 3k + 4 = 2k - 5 \implies k = -9.\]

Plugging in \(-9\) for \(k,\) we see that \(f(3) = 3(-9) + 4 = -23.\) Furthermore, we verify that \(\lim_{x \to 3} f(x) = -23,\) which matches the value of \(f(3).\) Because of this agreement, we conclude that, for \(k = -9,\) \(f(x)\) is continuous at \(x = 3.\)



EXAMPLE \(4\)
Let \( \displaystyle f(x)= \begin{cases} 3k x^3 + 2k^2 x + 15 &x \lt 1 \\[2ex] 8kx - 18k x^2 &x \geq 1. \end{cases} \)

Find the values of \(k\) for which \(f(x)\) is continuous at \(x = 1.\)

For this problem, we notice that \[ \begin{aligned} f(1) &= 8k(1) - 18k(1)^2 \\ &= 8k - 18k \\ &= -10k. \\ \end{aligned} \] Therefore, in order for \(f(x)\) to be continuous at \(x = 1,\) \(\lim_{x \to 1} f(x)\) must equal \(-10k.\)

To evaluate the limit, we will consider the one-sided limits and set them equal to each other. For the left-hand-side limit (\(x \lt 1\)), we get \[ \begin{aligned} \lim_{x \to 1^-} f(x) &= 3k(1)^3 + 2k^2(1) + 15 \\ &= 2k^2 + 3k + 15. \\ \end{aligned} \] For the right-hand-side limit (\(x \gt 1\)), we get \[ \begin{aligned} \lim_{x \to 1^+} f(x) &= 8k(1) - 18k(1)^2 \\ &= 8k - 18k \\ &= -10k. \\ \end{aligned} \] Setting these one-sided limits equal to each other and solving for \(k\) give \[ \begin{aligned} 2k^2 + 3k + 15 &= -10k \\ 2k^2 + 13k + 15 &= 0 \end{aligned} \] \[k = -5 \quad \textrm{and} \quad k = -\frac{3}{2}.\]

Therefore, \(k = -5\) and \(\displaystyle k = -\frac{3}{2}\) permit \(f(x)\) to be continuous at \(x = 1.\)


We can utilize limit properties, discussed in 1.3, to prove various continuity properties. We can show that if \(f(x)\) and \(g(x)\) are continuous at \(x = a,\) then \((f + g)(x),\) \((f - g)(x),\) \((fg)(x)\), and (for \(g(x) \ne 0\)) \((f/g)(x)\) will all be continuous at \(x = a.\)

PROOF If \(f(x)\) and \(g(x)\) are both continuous at \(x = a,\) then \(\lim_{x \to a} f(x)\) and \(\lim_{x \to a} g(x)\) exist and equal \(f(a)\) and \(g(a),\) respectively. Therefore, \[ \begin{aligned} \lim_{x \to a} \left(f + g \right)(x) &= \lim_{x \to a} \left[f(x) + g(x) \right] \\ &= \lim_{x \to a} f(x) + \lim_{x \to a} g(x) \\ &= f(a) + g(a). \end{aligned} \] \((f + g)(a)\) is continuous if \((f + g)(a)\) is defined and \(\lim_{x \to a} \left(f + g \right)(x) = (f + g)(a).\) Because these conditions are satisfied, we have established that \((f + g)(x)\) is continuous at \(x = a.\) \[\tag*{$\blacksquare$}\]

Similarly, \[ \begin{aligned} \lim_{x \to a} \left(f - g \right)(x) &= \lim_{x \to a} \left[f(x) - g(x) \right] \\ &= \lim_{x \to a} f(x) - \lim_{x \to a} g(x) \\ &= f(a) - g(a). \\ \end{aligned} \\ \] \[ \therefore (f - g)(x) \, \, \textrm{is continuous at} \, \, x = a. \] \[\tag*{$\blacksquare$}\] \[ \begin{aligned} \lim_{x \to a} \left(fg \right)(x) &= \lim_{x \to a} \left[f(x) \cdot g(x) \right] \\ &= \lim_{x \to a} f(x) \cdot \lim_{x \to a} g(x) \\ &= f(a) \cdot g(a). \\ \end{aligned} \\ \] \[ \therefore (fg)(x) \, \, \textrm{is continuous at} \, \, x = a. \] \[\tag*{$\blacksquare$}\] \[ \begin{aligned} \lim_{x \to a} \left(\tfrac{f}{g}\right)(x) &= \lim_{x \to a} \frac{f(x)}{g(x)}, \lim_{x \to a} g(x) \ne 0 \\ &= \frac{\lim_{x \to a} f(x)}{\lim_{x \to a} g(x)} \\ &= \frac{f(a)}{g(a)}. \\ \end{aligned} \\ \] \[ \therefore \left(\tfrac{f}{g}\right)(x) \, \, \textrm{is continuous at} \, \, x = a, \, \textrm{provided that} \, \, g(a) \ne 0. \] \[\tag*{$\blacksquare$}\]


THEOREM \(2\): CONTINUITY OF COMBINATIONS OF FUNCTIONS

If \(f(x)\) and \(g(x)\) are both continuous at \(x = a,\) then the following combinations of \(f(x)\) and \(g(x)\) are also continuous at \(x = a.\)

  1. \((f + g)(x)\)
  2. \((f - g)(x)\)
  3. \((fg)(x)\)
  4. \((f/g)(x)\)

Using these limit properties, we can formally justify a fact we know from prior experience: polynomials are continuous for all real numbers.

PROOF If \(p(x)\) is a polynomial, then it is in the form \[ a_1 x^n + a_2 x^{n - 1} + a_3 x^{n - 2} + \cdots + a_{n - 1} x + a_n, \] where \(\left\{a_1, a_2, a_3, \dots, a_n\right\}\) is a set of constants. Therefore, if \(\lim_{x \to a} p(x) = p(a)\) for \(-\infty \lt a \lt \infty,\) then we can establish that \(p(x)\) is continuous for all \(x.\) Taking the limit, we get \[ \begin{aligned} \lim_{x \to a} \left(a_1 x^n + a_2 x^{n - 1} + a_3 x^{n - 2} + \cdots + a_{n - 1} x + a_n\right) \\ = \lim_{x \to a} \left(a_1 x^n \right) + \lim_{x \to a} \left(a_2 x^{n - 1} \right) + \lim_{x \to a} \left(a_3 x^{n - 2} \right) + \cdots + \lim_{x \to a} \left(a_{n - 1} x \right) + \lim_{x \to a} \left(a_n \right). \\ \end{aligned} \]

\((1)\)

Recall the property that \(\displaystyle \lim_{x \to a} cf(x) = c \lim_{x \to a} f(x),\) where \(c\) is a constant. Additionally, \(\displaystyle \lim_{x \to a} x = a\) and \(\displaystyle \lim_{x \to a} f(x) g(x) = \lim_{x \to a} f(x) \cdot \lim_{x \to a} g(x),\) given that \(f(x)\) and \(g(x)\) are both continuous at \(x = a\) (Theorem \(2,\) III.). We know that \(f(x) = x\) is continuous for all \(x.\) Therefore, we see that \[ \begin{aligned} \lim_{x \to a} \left(a_1 x^n \right) &= a_1 \lim_{x \to a} \left( x^n \right) \\ &= a_1 \lim_{x \to a} \Big(\underbrace{x \cdot x \cdot x \cdots \cdot x}_{n \, \textrm{times}} \Big) \\ &= a_1 \underbrace{\lim_{x \to a} x \cdot \lim_{x \to a} x \cdots \cdot \lim_{x \to a} x}_{n \, \textrm{times}}, \\ &= a_1 \underbrace{a \cdot a \cdots \cdot a }_{n \, \textrm{times}} \\ &= a_1 a^n, \end{aligned} \] and this pattern is applicable to every term of the polynomial.

\((2)\)

Therefore, \[\lim_{x \to a} p(x) = p(a),\] which states that \(p(x)\) is continuous for all \(x.\)

\((3)\)

\[\tag*{$\blacksquare$}\]

Now that we have proven that polynomials are continuous, we can establish continuity for rational functions in their respective domains. Rational functions are composed of a quotient of polynomials in the form \(p(x)/q(x).\) We understand that if \(p(x)\) and \(q(x)\) are both continuous, then the quotient \(p(x)/q(x)\) is also continuous, provided that \(q(x) \ne 0\) (Theorem \(2,\) IV.).


EXAMPLE \(5\)

For each of I.–III., determine the intervals on which the function is continuous.

  1. \(f(x) = x^4 - 13x^2 - 8\)
  2. \(\displaystyle g(x) = \frac{x^2 - 4x + 5}{(x - 9)(x - 7)}\)
  3. \(\displaystyle h(x) = \frac{8 - x}{4 + x}\)

  1. \(f(x)\) is a fourth-degree polynomial, which is continuous for all \(x.\) Therefore, the interval is \((-\infty, \infty).\)

  2. \(g(x)\) is a rational function, which is continuous for all \(x\) for which the denominator does not equal \(0.\) In this case, \(x = 7\) and \(x = 9\) make the denominator of \(g(x)\) equal \(0,\) so we exclude them. Thus, the interval of continuity is \((-\infty, 7) \cup (7, \infty).\)

  3. We notice that, for \(x = -4,\) the denominator of \(h(x)\) equals \(0.\) Thus, the interval of continuity is all real numbers except \(x = -4,\) \((-\infty, -4) \cup (-4, \infty).\)

Let us now determine the continuity of other functions. We will work with combinations of functions, whose properties of continuity we established with Theorem \(2,\) and composite functions, which are in the form \(f(g(x)).\) For the continuity of this composite function at \(x = a,\) we must establish that \[\lim_{x \to a} f(g(x)) = f(g(a)).\] If \(g(x)\) is continuous at \(x = a\) and \(f(x)\) is continuous at \(x = g(a),\) then \[\lim_{x \to a} f(g(x)) = f \left(\lim_{x \to a} g(x)\right) = f(g(a)),\] which satisfies our criteria for continuity.


THEOREM \(3\): CONTINUITY OF COMPOSITE FUNCTIONS

If \(g(x)\) is continuous at \(x = a\) and \(f(x)\) is continuous at \(x = g(a),\) then \(f(g(x))\) is continuous at \(x = a.\)


Consider, as an example, the function \(h(x) = \ln(x - 2).\) This function is in the form \(f(g(x)),\) where \(f(x) = \ln(x)\) and \(g(x) = x - 2.\) We know that \(\ln(x)\) is continuous for \(x \gt 0\) (logarithms are only defined for positive inputs.) Therefore, the nested function, \(g(x) = x - 2,\) must follow this restriction and be greater than \(0.\) \[ \begin{aligned} g(x) \gt 0 \\ x - 2 \gt 0 \\ x \gt 2. \\ \end{aligned} \] \(g(x) = x - 2\) is continuous for all \(x,\) and \(f(x) = \ln(g(x))\) is continuous for \(g(x) \gt 0,\) so \(\ln(x - 2)\) is continuous for all \(x \gt 2,\) as given by Theorem \(3.\)


EXAMPLE \(6\)

For each of I.–VI., determine the intervals on which the function is continuous.

  1. \(f(x) = \sin(x) + x^2\)
  2. \(\displaystyle g(x) = \frac{\cos(x) + x}{\ln(x)}\)
  3. \(\displaystyle h(x) = \frac{\tan(x)}{x + 2}\)
  1. \(\displaystyle i(x) = x^4 \sin(x) \)
  2. \(\displaystyle j(x) = e^{\sqrt{3 - x}}\)
  3. \(\displaystyle k(x) = \frac{x\sec(x)}{\sqrt{x - 5}}\)

  1. \(\sin(x)\) and \(x^2\) are both continuous for all \(x.\) Therefore, by I. of Theorem \(2,\) the sum \(f(x) = \sin(x) + x^2\) is also continuous for all \(x.\)

  2. The numerator of \(g(x),\) \(\cos(x) + x,\) is a sum of two continuous functions \(\cos(x)\) and \(x\) and is thus continuous for all \(x\) (I. of Theorem \(2\)). Conversely, the denominator, \(\ln(x),\) is only continuous for \(x \gt 0.\) Therefore, for \(x \gt 0,\) the numerator and denominator are both continuous—and by IV. of Theorem \(2,\) so is \(g(x).\)

  3. We seek to find the values of \(x\) for which the numerator and denominator of \(h(x)\) are both continuous (and the denominator \(\ne 0\)) since we can then establish continuity for \(h(x)\) by property IV. of Theorem \(2.\) We know that \(\tan(x)\) is continuous for all \(x\) except \(\pm \pi/2, \pm 3 \pi/2, \pm 5 \pi/2, \dots.\) We also understand that, at \(x = -2,\) the denominator of \(h(x)\) evaluates to \(0.\) Therefore, the values of \(x\) for which \(h(x)\) is discontinuous include \(x = -2\) and \(x = \pi/2 \pm \pi n, n = 0, \pm 1, \pm 2, \dots.\)

  4. \(x^4\) and \(\sin(x)\) are both continuous for all \(x.\) Therefore, by III. of Theorem \(2,\) the product \(i(x) = x^4 \sin(x)\) is also continuous for all \(x.\)

  5. We understand that \(e^x\) is continuous for all \(x.\) In this case, however, we have the composite function \(e^{\sqrt{3 - x}},\) and \(\sqrt{3 - x}\) is only continuous for \(x \leq 3.\) Therefore, \(e^{\sqrt{3 - x}}\) is continuous for all \(x \leq 3.\)

  6. Rewriting \(k(x)\) as \(\displaystyle \frac{x}{\cos(x) \sqrt{x - 5}},\) we want to find the values of \(x\) for which the numerator and denominator are both continuous (and for which the denominator is not \(0,\) of course). We notice that the numerator, \(x,\) is continuous for all values of \(x.\) On the denominator, \(\cos(x)\) is also continuous for all \(x,\) but it cannot equal \(0,\) which it does at \(x = \pm \pi/2, \pm 3 \pi/2, \pm 5 \pi/2, \dots.\) We will therefore exclude those points. \(\sqrt{x - 5}\) is continuous and positive for all \(x \gt 5,\) so—for \(k(x)\) to be continuous—we are confined to \(x\)-values greater than \(5\) such that \(\cos(x) \ne 0.\) Thus, \(k(x)\) is continuous for \(x \gt 5, x \ne \frac{5\pi}{2} + \pi n, n = 0, \pm 1, \pm 2, \dots.\)



SECTION SUMMARY

  • If \(\displaystyle \lim_{x \to a} f(x) = f(a),\) then \(f(x)\) is continuous at \(x = a.\)
  • To determine whether \(f(x)\) is continuous at \(a,\) \((1)\) ensure that \(f(a)\) is defined, \((2)\) find \(\displaystyle \lim_{x \to a} f(x),\) and \((3)\) verify that \(\displaystyle \lim_{x \to a} f(x) = f(a).\)
  • \(f(x)\) is continuous over an interval if \(f(x)\) is continuous at all values of \(x\) inside the interval.
  • To check the continuity at an endpoint \(x = a,\) first ensure that \(f(a)\) is defined. If \(a\) is a left endpoint, verify that \(\displaystyle \lim_{x \to a^+} f(x) = f(a).\) If \(a\) is a right endpoint, verify that \(\displaystyle \lim_{x \to a^-} f(x) = f(a).\)
  • Polynomials are continuous for all real numbers. Rational functions are continuous in their domains.
  • If \(f(x)\) and \(g(x)\) are both continuous at \(x = a,\) then \((f + g)(x),\) \((f - g)(x),\) \((fg)(x),\) and (for \(g \ne 0\)) \((f/g)(x)\) are all continuous at \(x = a.\)
  • If \(g(x)\) is continuous at \(x = a\) and \(f(x)\) is continuous at \(x = g(a),\) then \(f(g(x))\) is continuous at \(x = a.\)
